# How to Cite

If you use this code or build upon this work, please cite:

## Paper Citation

### BibTeX

```bibtex
@article{chin2025hexagonal,
  title={Hexagonal Qutrit Quantum Error Correction: Matching Surface Code Thresholds with 20\% Resource Savings},
  author={Chin, Eddie},
  journal={[Journal Name]},
  volume={[Volume]},
  pages={[Pages]},
  year={2025},
  publisher={[Publisher]},
  doi={[DOI]},
  note={arXiv:2025.xxxxx}
}
```

### APA Style

Chin, E. (2025). Hexagonal qutrit quantum error correction: Matching surface code thresholds with 20% resource savings. *[Journal Name]*, *[Volume]*, [Pages]. https://doi.org/[DOI]

### MLA Style

Chin, Eddie. "Hexagonal Qutrit Quantum Error Correction: Matching Surface Code Thresholds with 20% Resource Savings." *[Journal Name]*, vol. [Volume], 2025, pp. [Pages].

### Chicago Style

Chin, Eddie. "Hexagonal Qutrit Quantum Error Correction: Matching Surface Code Thresholds with 20% Resource Savings." *[Journal Name]* [Volume] (2025): [Pages].
